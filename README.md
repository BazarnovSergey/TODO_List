### приложение "TODO список"

#### В этом проекте используются такие технологии как:

1. Spring boot 2.5.2
2. Thymeleaf 3.0.15
3. Bootstrap 4
4. Hibernate 5.6.11
5. PostgreSQL 14
6. Java 17


#### Для сборки проекта понадобятся:
1. JDK 17
2. Maven 3.8.5
3. PostgreSQL 14

#### Запуск проекта:
1. Загрузите проект на свой компьютер.
2. В PostgreSQL создайте базу данных с названием todo.
3. В корневой папке проекта выполните команду
```shell
mvn install
```
4. Для запуска проекта используйе команду
 ```shell
java -jar target/job4j_todo-1.0.jar
```
5. В браузере перейдите по ссылке http://localhost:8080/index

На главной странице отображается список всех заданий.

При переходе по ссылкам в таблице отображаются: все задания, только выполненные или только новые.
